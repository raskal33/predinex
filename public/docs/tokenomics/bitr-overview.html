PRIX Token Overview

PRIX is the native utility token of the Prixedict ecosystem, designed to align incentives between users, liquidity providers, and the platform. As an ERC-20 token on the Somnia network, PRIX serves as the backbone for staking rewards, fee reduction, and platform utility.

PRIX (Prixedict Token) is a utility token that powers the Prixedict ecosystem:

  • Staking Rewards: Earn revenue share from platform fees through tiered staking
  • Fee Discounts: Reduce trading fees for token holders
  • Premium Access: Unlock advanced features and analytics
  • Platform Utility: Required for various platform interactions

Stake PRIX to earn dual rewards:

  • Fixed APY: 6%, 12%, 18% base rates respectively
  • Revenue sharing: 10%, 30%, 60% of platform fees
  • Lock periods: 30/60/90 days with bonus APY
  • Dual token rewards: Earn both PRIX and BNB

How the Testnet Faucet Worksâ€‹

To participate in Prixedict's testnet and qualify for the mainnet airdrop, users can claim testnet PRIX through our faucet system:

Faucet Eligibilityâ€‹

  • 20,000 testnet PRIX distributed per eligible wallet
  • One claim per wallet - no multiple claims allowed
  • Activity requirement: Must interact with platform using BNB before claiming
    • Create a prediction pool, OR
    • Place a bet on an existing pool

Purpose of Testnet PRIXâ€‹

Testnet PRIX enables users to:

  • Test staking mechanisms across all three tiers
  • Participate in platform governance testing
  • Access premium features during testing phase
  • Build activity history for mainnet airdrop eligibility

Claiming Processâ€‹

  1. Use the platform first with BNB (create pool or place bet)
  2. Connect your wallet to the testnet faucet
  3. Claim 20,000 testnet PRIX (one-time only)
  4. Use the tokens to test staking and other features

Mainnet Transitionâ€‹

Testnet PRIX activity and holdings will be used to determine mainnet airdrop eligibility and amounts. See our Airdrop Documentation for complete details on the testnet-to-mainnet transition process.
